JavaScript学习笔记-1

2020-03-19

1. 简介

基于对象和事件驱动的语言，应用于客户端。

  • 基于对象：提供好了很多对象可以直接拿过来使用

  • 事件驱动：动态效果

  • 客户端：专指浏览器（b/s）

特点：

  • 交互性：信息的动态交互
  • 安全性：不能访问本地磁盘的文件
  • 跨平台性：能够支持JS的浏览器都可以运行

和Java的区别：

  • Java sun oracle，JS 网景

  • JS基于对象，Java面向对象

  • Java强类型，Js弱类型

    js: var i="10"; var i=123;

  • JS只需解析就可以执行，Java需要先编译再运行

组成：

  • ECMAScript：由ECMA制定的JS语句、语法等

    ECMA：欧洲计算机协会

  • BOM：浏览器对象模型 browser object model

  • DOM：文档对象模型 document object model

2. 与HTML的结合方式

  • 第一种：使用标签<script type="text/javascript"> js代码 </script>

  • 第二种：使用script标签引入外部js文件

    <script type="text/javascript" src="1.js"></script>

    该种方式再往script标签内写js代码也不会执行。

  • 全局变量：在script标签里面定义一个变量，这个变量在页面中js部分都可以使用(在另外的script标签里也可以使用)

11.String对象

  • 方法和属性

    方法:

    • 与html相关的方法

      • fontsize();//字符串大小，1-7

      • link();//字符串显示为超链接

    • concat();//连接字符串方法

    • split();//切分字符串，切分为数组

    • replace();//替换字符串，第一个参数是被替换的值，第二个参数是替换后的值

    • substr()和substring()

      substr(a,b);//从第a位开始向后截取b位

      substring(a,b)；//从第a位开始到b位结束但不包含第b位

12.Array对象

  • 创建数组

    • var arr1=[1,2,3];
    • var arr2=new Array(2);
    • var arr3=new Array(1,2,3);

  • 属性

  • length：数组的长度

  • 方法

    • concat();//连接数组

    • join();//根据指定的字符分割数组元素，默认，隔开

    • push();//向数组末尾添加元素，返回新长度

      arr1.push(arr2)，相当于把arr2当成一个数组元素传入，长度+1。

    • pop();//删除并返回最后一个元素

    • reverse();//颠倒数组中的元素顺序

13. Date对象

Java中获取当前时间：Date date=new Date();

SimpleDateFormat sdf=new SimpleDateFormat(“yyyy-MM-dd HH:mm:ss”)；sdf.format(date);

date.toLocaleString();

  • 获取当前时间

    var date=new Date()

  • 转换为本地时间的格式：date.toLocaleString();

  • 返回年份：date.getFullYear();

  • 返回月份：date.getMonth();//返回0到11，需要加一

  • 返回星期：date.getDay();//返回0到6，周日是0

  • 返回当前的日：date.getDate();//1到31

  • 获取当前的小时：date.getHours();//0到24

  • 分钟：date.getMinutes();

  • 秒:date.getSeconds();

  • 毫秒：date.getTime();//返回1970至今的毫秒数

    使用毫秒数处理缓存的效果。

16. bom对象（浏览器对象模型 browser object model）

有哪些对象？

  • navigator：Navigator 对象包含有关浏览器的信息。

    navigator.appName //返回浏览器名称

  • screen：Screen 对象包含有关客户端显示屏幕的信息。

  • location：Location 对象包含有关当前 URL 的信息。

    href属性：可设置或返回当前显示的文档的完整 URL。

    document.write(location.href);//返回当前地址

    location.href="url";//设置地址

    鼠标点击事件：onclick=“js方法”

  • history：请求的URL历史记录

    • 到访问的上一个页面

      history.back();

      history.go(-1);

    • 到访问的下一个页面

      history.forward();

      history.go(1);

  • window ✨

    是一个窗口对象，也是一个顶层对象，所有的bom对象都是在window里面操作的（包含navigator、location等）。

    属性：

    • opener：opener 属性是一个可读可写的属性，可返回对创建该窗口的 Window 对象的引用。

    方法：

    • onload():在网页加载完毕后立刻执行的操作，即当 HTML 文档加载完毕后，立刻执行某个方法

  • alert()：弹出框显示内容，简写alert();

  • confirm(); 弹出确认框，返回boolean值

    • window.prompt(text, default text);//输入对话框

      prompt("请输入：","0");

    • open(); 打开一个新的窗口

    open(“打开的新窗口的地址url”，“name，可选“，“窗口特征，比如窗口的宽度和高度”)

    • close();关闭窗口（浏览器兼容性较差）

    • 做定时器：

      • setInterval();按照指定的周期（以毫秒计）来调用函数或计算表达式。返回id值。

        setInterval("js代码",毫秒数)

        ``setInterval(“alert(123);”,3000);//每三秒alert123`

      • setTimeout();在指定的毫秒数后调用函数或计算表达式，只执行一次。

        返回id值。

        setTimeout("js代码",毫秒数)

      • clearInterval();清除setInterval()设置的定时器

      • clearTimeout();清除setTimeout()设置的定时器

17.dom对象（文档对象模型，document object model）

文档：超文本文档（超文本标记文档） html、xml

对象：提供了属性和方法

模型：使用属性和方法操作超文本标记型文档

可以使用js里面的dom里面提供的对象，使用这些对象的属性和方法，对标记型文档进行操作。

要想对标记型文档进行操作，首先需要对标记型文档里面的所有内容封装成对象，即把html里面的标签、属性、文本内容都封装成对象

然后再使用dom解析标记型文档html。解析过程：1.根据html的层级结构在内存中分配一个树形结构；2.把html的每个部分都封装成对象，包括document对象，element对象，属性对象，文本对象，以及所有对象的父类对象Node对象，当在子类对象中找不到合适方法时可以在Node对象中查找。

  • DHTML：很多技术的简称

    • html：封装数据
    • css：使用属性和属性值设置样式
    • dom：操作html文档(标记型文档)
    • JavaScript：专指js的语法语句（ECMAScript）

  • Document对象（document对象也是window对象的一部分）：整个文档

    每个载入浏览器的HTML文档都会成为Document对象。常用方法：

    • write()方法：向页面输出变量值或html代码

    • getElementById();通过id得到标签，返回对拥有指定id的第一个对象的引用

    • getElementsByName();返回带有指定名称的对象集合

    • getElementsByTagName();返回带有指定标签名称的对象集合

  • document.createElement(“标签名”) 创建标签

  • createTextNode(“内容”) 创建文本

  • Element对象

    要操作Element对象，必须要先获取Element。

    方法：

    • 获取属性对象：getAttributeNode(“属性名称”)

    • 获取属性里面的值：getAttribute(“属性名称”);

      alert(input1.class);无法获取，不能直接获取关键字，

      但alert(input1.getAttribute(“class”))可获取;

    • 设置属性里的值：setAttribute(“属性名称”,”属性值”);

      可以直接通过input1.属性名称=值来设定。

    • 删除属性里的值：removeAttribute(“属性名称”)

      不能删除value值

    • 在Element对象的范围里，查找其他子标签的唯一有效方法就是getElemensByTagName()

  • Node对象

    属性一：

    • nodeName：标签（大写标签名称）/属性（属性名称）/文本（#text）
    • nodeType：1/2/3
    • nodeValue：null/属性的值/文本内容

    属性二：

    专指标签之间的关系，不包括属性，属性是单独取的。

    • 父节点：如ul是li的父节点

      parentNode

    • 子节点：li是ul的子节点

      childNodes(得到所有的子节点，但兼容性差)

      firstElementChild：获取第一个子节点

      lastElementChild：获取最后一个子节点

  • 同辈节点：

    nextElementSibling：获取前一个节点

    previousElementSibling：获取后一个节点

  • 操作DOM树

    • appendChild方法：将子节点添加到末尾

      特点：类似于剪切粘贴的效果

    • insertBefore(newNode,oldNode)方法：在某个节点（old）之前插入新的节点(通过父节点调用)

      newNode：创建标签、创建文本、将文本添加至标签

    • removeChild方法：通过父节点删除

    • replaceChild(newNode,oldNode)方法：替换节点，通过父节点实现

    • cloneNode(boolean)方法：复制节点，true 返回一个副本

    操作dom树的总结：

    • 获取节点使用方法：

      document.getElementById();

      document.getElementsByName();

      document.getElementsByTagName();

      使用parentNode、firstElementChild、nextElementSibling等node属性

    • 插入节点的方法

      appendChild(); //末尾，剪切黏贴

      insertBefore(); //在某个节点之前加入,父节点

    • 删除节点的方法

      removeChild();//父节点

    • 替换节点的方法：

      replaceChild();//父节点
